Recipe Timer Demo

This demonstrates the timer functionality. Click on the timer buttons next to time durations:

Example Instructions:

  1. Preheat oven to 350°F
  2. Mix ingredients for 5 minutes
  3. Let dough rest for 30 minutes
  4. Bake for 45 minutes
  5. Cool for 10-15 minutes before serving

Features:

  • ✅ Automatically detects time durations in recipe instructions
  • ✅ Click timer button to start countdown
  • ✅ Play/pause functionality
  • ✅ Stop button to cancel timer
  • ✅ Audio notification when timer completes
  • ✅ Supports ranges (uses maximum time)
  • ✅ Multiple timers can run simultaneously
